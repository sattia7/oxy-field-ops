# AI Field Operations Training App

A bilingual (EN/AR) interactive training application for oil field operations powered by AI. Built as a single-file Progressive Web App (PWA).

## 🌐 Live Demo

Deployed automatically via Vercel on every push to `main`.

## 🚀 Features

- **Bilingual**: Full English + Arabic support with RTL layout
- **Interactive Missions**: 15 hands-on training missions
- **Gamification**: Points, badges, streaks, and leaderboards
- **Offline-First**: Works without internet after first load
- **Trainer Dashboard**: Group management and progress analytics
- **Responsive**: Mobile-first design for field use

## 📦 Deployment

This repository is configured for **automatic deployment to Vercel**:

1. Push to `main` → Vercel builds and deploys
2. Preview deployments for every Pull Request
3. Production URL: `https://oxy-fieldops.vercel.app`

## 🛠 Local Development

Simply open `index.html` in any modern browser. No build step required.
